<Section icon={Target} title="Local heave ho 2 gameshare Setup Guide">

Setting up a local **heave ho 2 gameshare** session is straightforward, but it requires a specific sequence of actions on both the host and the guest consoles. This method is ideal for situations where internet connectivity is limited or unavailable, as it creates a direct link between the hardware.

<Steps>
  <Step num="1" title="Initialize Host Session">
    On the primary console (the one that owns the game), navigate to the main menu and select the **GameShare** option. From there, choose **Local GameShare** to begin broadcasting a lobby signal.
  </Step>
  <Step num="2" title="Guest Console Connection">
    On the guest console, open the game (or the GameShare client). Select **GameShare**, then **Join GameShare**, and finally **Local User**. The console will begin searching for active local lobbies.
  </Step>
  <Step num="3" title="Account Selection">
    The guest must choose the local account they wish to use. Once selected, the host lobby should appear on the screen. Press the primary action button (usually 'A') to join the session.
  </Step>
  <Step num="4" title="Confirm and Launch">
    Once the guest appears in the host's lobby, the host can press Start. Note that in local mode, online features are disabled to maintain the direct hardware connection.
  </Step>
</Steps>

When playing locally, the latency is virtually non-existent, which is crucial for a game that relies so heavily on precise timing and physics. If you are trying to grab a friend's hand mid-air, every millisecond counts.

| Local Component | Host Requirements | Guest Requirements |
| :--- | :--- | :--- |
| **Software** | Full version installed | GameShare client or demo |
| **Connectivity** | Local Wireless enabled | Local Wireless enabled |
| **Profile** | Primary owner account | Any local user profile |
| **Distance** | Within 10-15 feet | Within 10-15 feet |

<Callout type="warning" title="Connection Tip">
If the guest console cannot find the host, ensure that both devices are not connected to a 5GHz Wi-Fi network that might interfere with the direct local wireless signal. Sometimes toggling Airplane Mode on and off can reset the search.

<Section icon={Shield} title="Troubleshooting and Performance Tips">

While the **heave ho 2 gameshare** system is remarkably stable, users may occasionally encounter connection issues or input lag, especially during online sessions. Maintaining a smooth experience is vital for a game where a single frame of lag can result in everyone falling into the abyss.

| Issue | Potential Cause | Recommended Solution |
| :--- | :--- | :--- |
| **Lobby Not Found** | Version Mismatch | Update both consoles to the latest patch |
| **Input Lag** | High Ping (Online) | Use a wired connection or 5GHz Wi-Fi |
| **Disconnects** | Signal Interference | Move consoles closer (Local) or reset router |
| **Invite Failed** | Privacy Settings | Ensure "Allow Game Invites" is enabled in OS |

If you are experiencing "ghosting" or stuttering during GameShare, it is often due to the host's upload speed. Since the host is essentially streaming the game data to the guest, a stable upstream connection is required. We recommend at least 5-10 Mbps upload speed for the best online GameShare experience in 2026.
