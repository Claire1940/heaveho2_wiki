モメンタム（勢い）を構築することは、**ヒーブホー 2 eShop**体験の核心です。プレイヤーが協力すれば、一人では不可能な障害物を乗り越えるのに十分な力を生み出すことができます。これには言葉によるコミュニケーションが必要であり、特にスピードが不可欠な「ショーダウン」対戦レベルでは重要になります。

| アクション | コントローラー入力 | 戦術的用途 |
| :--- | :--- | :--- |
| **左手で掴む** | L / ZL トリガー | 壁、オブジェクト、または仲間に固定する。 |
| **右手で掴む** | R / ZR トリガー | 次の足場に手を伸ばす、または友人を捕まえる。 |
| **体を回転させる** | 左アナログスティック | 遠心力を利用して長いジャンプを行う。 |
| **仲間を投げる** | 掴む + スイング + 離す | 危険な障害物を越えて仲間を発射する。 |
| **登る** | L/Rを交互に掴む | 棒、ロープ、または壁を垂直に移動する。 |

<Callout type="warning" title="重要">
両手を同時に離すと、即座に落下します。**ヒーブホー 2 eShop**版では、重力が最大の敵です。もう一方の腕を動かす前に、必ずグリップインジケーターを目視で確認してください。

<Section icon={Users} title="Nintendo Switch 2 ゲームシェアとマルチプレイヤー設定">

Nintendo Switch 2向けの**ヒーブホー 2 eShop**リリースの際立った機能の一つは、ゲームシェア（GameShare）機能です。これにより、ゲームの所有者一人がセッションをホストし、ゲームを所有していない他のプレイヤーを最大3人まで招待できます。これはGameChatシステムを通じて、ローカルでもオンラインでも機能します。

<Steps>
  <Step num="1" title="ゲームシェアの開始">
    ホストはeShopで購入したヒーブホー 2のデジタル版を所有している必要があります。マルチプレイヤーメニューを開き、「ゲームシェアセッションをホスト」を選択します。
  </Step>
  <Step num="2" title="参加者の接続">
    近くのプレイヤー（ローカル）またはフレンドリストのフレンド（オンライン）がセッションに参加できます。オンラインプレイの場合、すべての参加者が有効なNintendo Switch Onlineメンバーシップを持っていることを確認してください。
  </Step>
  <Step num="3" title="ゲームモードの選択">
    8つのワールドがある協力キャンペーン、または56レベルの対戦ショーダウンモードから選択します。セッション中に行われたすべての進行状況は、ホストのプロファイルに保存されます。
  </Step>
  <Step num="4" title="GameChatでの連携">
    内蔵のGameChatを使用してスイングを同期させます。「掴んで」「スイング」「離して」といったシンプルなコマンドが、後半のレベルで成功するために不可欠です。
  </Step>
</Steps>

| マルチプレイヤーモード | プレイヤー数 | プラットフォーム | 必要条件 |
| :--- | :--- | :--- | :--- |
| **オンライン協力プレイ** | 2-4人 | 全プラットフォーム | インターネット + サブスク (コンソール) |
| **ローカル協力プレイ** | 2-4人 | 全プラットフォーム | 複数のコントローラー |
| **Steam Remote Play** | 2-4人 | PCのみ | ホストが所有 + 安定した通信環境 |
| **ローカル ゲームシェア** | 2-4人 | Switch 2のみ | ソフト1本 + 複数のSwitch 2本体 |
| **オンライン ゲームシェア** | 2-4人 | Switch 2のみ | ソフト1本 + GameChat + NSO |

ローカルでプレイするときは、画面に表示されるキャラクターと同じ順序で座るようにしましょう。これにより、物理演算が混乱している最中でも「左に投げて」といった指示が処理しやすくなります。
</Callout>

<Section icon={Map} title="8つのテーマワールドの探索">

| ワールド名 | 主要ギミック | 主な目的 |
| :--- | :--- | :--- |
| **フライング・ハイ** | スキーリフト＆ゴンドラ | 氷の天候下で移動する輸送プラットフォームを操作。 |
| **マナー・ミステリーズ** | 隠された鍵と錠 | 罠を避けながら出口を開ける鍵を見つける。 |
| **スペースバウンド・スクワッド** | 無重力物理 | 宇宙船を操縦し、無重力空間で勢いを管理。 |
| **シェフズ・オーダー** | 食材の積み上げ | 動く皿の上にかさばる食材を運んで積み上げる。 |
| **シップレック** | 水と浮遊物 | 浮力のあるアイテムを使用して隙間を渡り、ゴールを目指す。 |
| **トランスフォラボ** | インタラクティブ・デバイス | 研究所の機械を操作して道を切り開く。 |
| **カスケード・ショー** | 動く要素 | 変化するステージで精密なジャンプを連携させる。 |
| **メディーバル・トラベルズ** | 玉座の輸送 | 王様を玉座から転げ落とすことなく安全に運ぶ。 |

各ワールドには約8～10のレベルがあり、合計で72の協力ステージが存在します。ワールドを完了した後は、隠されたシルエット、謎の手、フォトチャレンジなどのオプションの収集品を探しに戻ることができます。
