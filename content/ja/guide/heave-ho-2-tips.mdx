<Section icon={Target} title="複雑なパズルステージの攻略">

ヒーブホー 2 には、単なるスイング以上の能力が求められる「パズルステージ」が登場します。これらのステージでは、鍵やティキ像などの特定のアイテムを見つけ、指定されたスロットまで運ぶ必要があります。これらのパズルでは、周囲へのグリップを維持しながらオブジェクトを操作する能力が試されます。

<Steps>
  <Step num="1" title="目的を確認する">
    クエスチョンマークのアイコンや空のスロットを探しましょう。多くのステージでは、「本物」の鍵が複数の偽物の中に隠されています。
  </Step>
  <Step num="2" title="進路を確保する">
    目的のアイテムを運ぶ前に、石のブロックや揺れるハンマーなどの障害物をあらかじめどけておきましょう。
  </Step>
  <Step num="3" title="輸送チェーンを作る">
    一人がアイテムを持つことに専念し、他のプレイヤーはそのプレイヤーを運ぶことに集中しましょう。スイングしながらアイテムを保持するのは非常に困難です。
  </Step>
  <Step num="4" title="挿入を実行する">
    アイテムをスロットまで運びます。一部のスロットでは、特定の向き（例：ティキの頭を直立させる）が必要な場合があります。
  </Step>
</Steps>

「鍵パズル」は、狭い隙間の連続に鍵を通さなければならない悪名高いチャレンジです。ここで「放り投げ」メカニクスを使うのは危険です。代わりに、制御された「手渡し」パスを使用して、鍵が到達不能な場所に落ちないようにしましょう。

| パズルタイプ | 主要なメカニクス | 難易度 |
| :--- | :--- | :--- |
| **鍵と錠前** | 精密なパスとグリップ管理 | 中 |
| **ティキの配置** | オブジェクトの向きと重量物の運搬 | 高 |
| **スイッチシーケンス** | 複数のレバーの同時起動 | 中 |
| **隠し通路** | 環境の観察とブラインドジャンプ | 高 |

<Callout type="success" title="パズルの近道">
複数の鍵があるパズルステージでは、かすかな光や質感の違いなど、微妙な視覚的ヒントを探してください。これらは多くの場合「本物」の鍵を示しており、すべての偽物をテストする時間を節約できます。
</Callout>

</Section>

<Section icon={ChefHat} title="料理とインタラクションステージのマスター">

続編における最もユニークな追加要素の一つが「シェフモード」ステージです。これらのステージでは、レシピを完成させるためにレタス、チーズ、チキンなどの食材を特定の順序で組み立てる必要があります。これには高度な空間認識能力と、複数のアイテムを落とさずに扱うスキルが必要です。

**料理ステージの作業手順:**
1. **下のバンズ**: 調理プレートの上に土台を固定します。
2. **メインのタンパク質**: チキンやパティを持ってきて、その上に重ねます。
3. **トッピング/付け合わせ**: 要求に応じてレタス、チーズ、ソースを追加します。
4. **上のバンズ**: サンドイッチを完成させます。
5. **ベル**: ステージをクリアするために、一人がサービスベルまで届いて鳴らさなければなりません。

| 食材 | 重量/物理特性 | 扱いやすさ |
| :--- | :--- | :--- |
| **バンズ** | 軽い / 摩擦が高い | 容易 |
| **チーズ/レタス** | 非常に軽い / 浮く | 普通 |
| **チキン/パティ** | 重い / 滑りやすい | 難しい |
| **ソース/マスタード** | 流体 / 汚れやすい | 非常に難しい |

<Callout type="info" title="食材の管理">
食材がプラットフォームから落ちてしまった場合、通常は元の場所にリスポーンします。落ちていくレタスを追って奈落に飛び込まないようにしましょう。安全を確保してリスポーンを待つのが得策です。
</Callout>

</Section>

<Section icon={Rocket} title="宇宙の航行と無重力戦略">

ヒーブホー 2 の宇宙ステージでは、無重力メカニクスとロケットの操縦が導入されます。常に下向きに働く重力がなくなるため、ゲームの根本的な物理法則が変わります。代わりに、慣性とレバーを駆使して宇宙を航行しなければなりません。

**宇宙ステーション操縦ガイド:**
- **左レバー**: 左のスラスターを制御（船体を右に傾ける）。
- **右レバー**: 右のスラスターを制御（船体を左に傾ける）。
- **下レバー**: メイン推進力を制御。
- **連携**: 二人がサイドレバーを担当し、三体目がメインエンジンを管理するのが理想的です。

| 宇宙の要素 | 物理的な影響 | 航行戦略 |
| :--- | :--- | :--- |
| **無重力** | 無限の慣性 / 落下ダメージなし | 壁を蹴って進む。何もない空間に漂い出ないように注意 |
| **スラスターレバー** | 高トルク / 急激な回転 | 短くリズミカルに引いて直進を維持する |
| **小惑星** | 衝突の危険 | 掴まることで不要な慣性を止めることができる |
| **エアロック** | 進行のゲート | 二人のプレイヤーが同時にレバーを引く必要がある |

<Callout type="tip" title="無重力での移動">
無重力セクションでは、腕を円を描くように素早く動かすことで、空中で「泳ぐ」ことができます。速度は遅いですが、掴める場所から離れて漂ってしまった時の救済策になります。
</Callout>
