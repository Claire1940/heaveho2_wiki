<Section icon={Hand} title="基本操作とメカニズムの基礎">

物理演算ベースのカオスな世界へようこそ。この**ヒーブホー 2 初心者ガイド**では、イーグルのように舞うか、ジャガイモのように落下するかを左右する基本操作を分解して解説します。このゲームの操作体系は驚くほどシンプルで、主に2つのボタン（左右のトリガーまたはバンパー）に集中します。これらはキャラクターの左手と右手にそれぞれ対応しています。

左右の手を独立して動かすコツを掴むことが、初心者にとって最初の大きな壁です。標準的なプラットフォーマーとは異なり、移動はすべて「どこを掴み、どう体重を振るか」によって決まります。

| 操作入力 | 実行されるアクション | 最適な使用例 |
| :--- | :--- | :--- |
| **左トリガー (L2/LT)** | 左手で掴む | 壁、床、仲間にしがみつく |
| **右トリガー (R2/RT)** | 右手で掴む | 体勢の安定化や手を伸ばす |
| **左アナログスティック** | キャラクターの体を回転 | 勢いをつける、スイングの狙いを定める |
| **両トリガー** | 両手で掴む | 狭い足場での最大限の安定 |
| **スティック + 離す** | 発射/投げる | 大きな溝を越える、ゴールに到達する |

<Callout type="info" title="安定化のヒント">
仲間にぶら下がる時は、可能な限り**両手**を使うようにしましょう。これによりキャラクターが制御不能に回転するのを防ぎ、上にいる人がスイングの狙いを定めやすくなります。
</Callout>

<Section icon={Move} title="慣性とスイング物理のマスター">

**ヒーブホー 2 初心者ガイド**における戦略の核心は「勢い（慣性）」です。あなたは実質的に、紐についた重りです。効果的に動くには、重力と遠心力がキャラクターの「ジャガイモのような」物理体とどう相互作用するかを理解する必要があります。

片手でぶら下がっているとき、キャラクターは自然に垂れ下がります。アナログスティックを円を描くように動かすことで、体を回転させることができます。十分な速度がついたら、ターゲットに応じて「12時」または「2時」の方向で掴みボタンを離すと、空中へ飛び出します。

| スイングの段階 | プレイヤーの操作 | 物理的な結果 |
| :--- | :--- | :--- |
| **初期吊り下げ** | 片方のトリガーを保持 | わずかに揺れる静止状態 |
| **溜め** | スティックを素早く回転 | 遠心力と速度を蓄積 |
| **弧の頂点** | 発射角度を見極める | ジャンプのための最大ポテンシャルエネルギー |
| **解放** | トリガーを離す | 勢いを飛行エネルギーに変換 |
| **着地** | 反対の手を準備 | 落下を防ぐため即座に掴む |

<Callout type="warning" title="慣性に関する注意">
回転させすぎると「目が回った」状態になり、一時的に操作が難しくなります。激しくスティックを回すのではなく、スムーズで一定の回転を心がけましょう。
</Callout>

<Steps>
  <Step num="1" title="アンカー（固定）">
    行きたい方向に近い方の手で、硬い表面をしっかり掴みます。
  </Step>
  <Step num="2" title="溜め">
    スイングしたい方向にアナログスティックを回します。右に振るなら時計回り、左なら反時計回りです。
  </Step>
  <Step num="3" title="リリースポイント">
    キャラクターの体が目標地点の少し上を向いたタイミングでトリガーを離します。物理法則により、軌道はわずかに下がります。
  </Step>
  <Step num="4" title="キャッチ">
    新しい足場に近づいたら、両方のトリガーを押し続けます。これにより、どちらの手が先に触れても即座にロックされます。
  </Step>
</Steps>

<Section icon={Users} title="高度な協力チェーン戦略">

『ヒーブホー 2』は協力プレイを前提に設計されています。ソロプレイも可能ですが、2〜4人で協力して環境パズルを解くときに真価を発揮します。「人間チェーン」はゲームで最も象徴的な動きであり、後半の複雑なステージをクリアするために不可欠です。

チェーンでは、一番上のプレイヤーが「アンカー（錨）」、一番下が「スインガー（振り子）」となります。中間のプレイヤーは「リンク（鎖）」です。グループ全員が奈落の底へ落ちないよう、それぞれの役割に責任があります。

| チェーンの役割 | 主な責任 | 重要なスキル |
| :--- | :--- | :--- |
| **アンカー** | 壁にしがみつき続ける | 高い握力維持力と忍耐 |
| **リンク** | 接続を維持する | 仲間の両手を掴み続ける |
| **スインガー** | 勢いを生み出す | 正確なリリースタイミング |
| **運び役** | ステージのコインを持つ | スイングしながら片手での操作 |

<Callout type="tip" title="プロの戦略">
チェーンが揺れすぎた場合は、アンカーは動きを完全に止めてください。物理的な挙動が落ち着くのを待ってから次のスイングを試みましょう。視覚的な合図に反応するよりも、言葉でのコミュニケーションの方が早いです。
</Callout>

**チェーン効率の目安:**
- **2人チェーン**: 精度と狭い隙間に最適。勢いは弱いがコントロールしやすい。
- **3人チェーン**: 「スイートスポット」。ほとんどの隙間に対応でき、大きなスイングに必要な重さも確保できる。
- **4人チェーン**: 最大のリーチだが非常にカオス。全員の体重を支える非常に強力なアンカーが必要。

</Section>

<Section icon={Gem} title="コイン、収集アイテム、カスタマイズ">

『ヒーブホー 2』の各レベルにはゴールデンコインが配置されています。これらを集めることが、新しいスキン、帽子、キャラクターボイスをアンロックする主な方法です。しかし、コインは片手を塞いでしまうため、難易度が大幅に上がります。

コインを拾うと、残りの片手だけで壁や仲間を掴まなければなりません。これにより、登りやスイングが格段に危険になります。

| 収集アイテム | 難易度変化 | 報酬タイプ |
| :--- | :--- | :--- |
| **標準コイン** | 難易度 +50% | ガチャマシン用のトークン1枚 |
| **隠しトークン** | 難易度 +75% | レア衣装のアンロック |
| **タイムトライアルスター** | 高速クリアが必要 | 称号エンブレム |
| **シークレットスキン** | 環境パズル | ユニークなキャラクターモデル |

<Callout type="success" title="コインのヒント">
コインはプレイヤー間で「受け渡し」が可能です。現在の運び役が苦戦している場合は、仲間が直接手からコインを奪い取ることができます。これにより、重荷を交代で担うことができます。
</Callout>

<Section icon={Settings} title="難関ステージでの生存のコツ">

ステージが進むにつれ、動くノコギリ、崩れる足場、滑る氷などのギミックが登場します。これらのエリアでの成功には、単なるスイング以上の「環境への意識」が必要です。

重要な機能の一つに「アシスト」の引き寄せがあります。仲間が落ちたり動けなくなったりした場合、ゴールにいるプレイヤーがロープや引き寄せメカニズムを作動させて助けられることがあります。不必要なリスタートを避けるため、利用可能なツールはすべて使いましょう。

| ギミックの種類 | プレイヤーへの影響 | 最善の対策 |
| :--- | :--- | :--- |
| **トゲ** | 即死 / リセット | ゆっくりと慎重に移動。無茶なスイングは厳禁 |
| **電撃ビーム** | 一時的な麻痺 | タイミング重視。サイクルを待って移動 |
| **動くノコギリ** | 即死 | 経路を予測し、隙を見てスイング |
| **氷の表面** | 摩擦 / グリップゼロ | 仲間の体をアンカーとして利用する |
| **風の流れ** | キャラクターを押し流す | 位置を維持するために風に逆らうように傾ける |

<Callout type="warning" title="デスペナルティ">
ほとんどのモードでは、一人が死ぬとチーム全員が最後のチェックポイントからやり直しになります。仲間の命を自分の命と同じように大切に扱ってください。
</Callout>
