<Section icon={Map} title="スペースバウンド・スクワッド：ワールド概要">

『Heave Ho 2（ヒーブホー 2）』は2026年7月16日にリリースされ、PC、Nintendo Switch、そしてNintendo Switch 2に、物理演算に基づいたカオスな協力プレイの新たな次元をもたらしました。全8つのテーマワールドの中でも、ワールド3「スペースバウンド・スクワッド」は、無重力環境のナビゲートと共有の**ヒーブホー 2 宇宙船**のマスターという、プレイヤーの連携を試す究極のテストを導入しています。このワールドでは、掴む、離す、振るといったすべての動作が軌道に多大な影響を与える無重力物理が導入され、従来の移動ルールが完全に覆されます。

この宇宙の冒険では、プレイヤーは協力して無重力の障害物コースを進み、回転する宇宙デブリを避け、共有のロケット船を操縦しなければなりません。初期の山や屋敷のレベルとは異なり、重力が安全な場所へと引き下げてくれることはありません。代わりに、ジャンプをミスすれば、チームメイトが助けの手を差し伸べない限り、虚空へと際限なく漂い続けることになります。

| レベルタイプ | 主要メカニクス | 主な目的 |
| :--- | :--- | :--- |
| 無重力ナビゲーション | 無重力ドリフト、手動グリップ | 全プレイヤーを安全に出口ゾーンまで導く |
| 宇宙船の操縦 | 連携したスラスター操作、ステアリング | チームの乗り物を狭い隙間に通す |
| 動く障害物 | 回転するデブリ、レーザーの危険 | タイミングを合わせてレーザーを回避する |
| 救助ミッション | アンカーの喪失、長いチェーンリーチ | 手を繋いで宇宙を漂うプレイヤーを回収する |

これらの宇宙の試練を生き抜くために、プレイヤーは従来の「振る」メカニクスから、慣性を利用した「推進」へと意識を切り替える必要があります。壁を蹴るたびに、別の表面を掴むか、チームメイトがアンカーになるまで、あなたはどこまでも進み続けます。

<Callout type="tip" title="無重力ドリフトの制御">
無重力レベルでは、慣性が最大の課題です。全員が同時にアンカーポイントを離さないようにしましょう。さもないと、チーム全体が宇宙の危険地帯へと無力に漂流してしまいます。他のメンバーが次の道を探っている間、少なくとも1人は固い壁にアンカーとして留まってください。
</Callout>

<Section icon={Target} title="ヒーブホー 2 宇宙船のマスター：協力操作">

ワールド3の目玉は、協力型の乗り物シーケンスです。**ヒーブホー 2 宇宙船**は1人のパイロットによって制御されるのではなく、チーム全員による物理的な操作を必要とする複数のインタラクティブなハンドルを備えています。船を移動、回転、操縦させるには、プレイヤーが特定のスラスターレバーを掴み、キャラクターの腕を同期したパターンで回転させる必要があります。

明確なコミュニケーションがなければ、船は制御不能に陥って激しく回転し、障害物に衝突して進行状況がリセットされてしまいます。船内での自分の役割を理解することが、打ち上げ成功への第一歩です。

| 役割 | 必要な入力 | 主な責任 |
| :--- | :--- | :--- |
| パイロット（左スラスター） | アナログスティック回転 ＋ 左グリップ | 左側の推進と反時計回りの回転を制御 |
| 副パイロット（右スラスター） | アナログスティック回転 ＋ 右グリップ | 右側の推進と時計回りの回転を制御 |
| 安定担当 | 船体への交互グリップ | 船のバランスを保ち、激しい回転を防ぐ |
| ナビゲーター | 自由移動、アクティブスキャン | 移動中に鍵やターゲットを掴む |

スペースバウンド・スクワッドの狭い通路をうまく通り抜けるには、以下のステップバイステップの操縦手順に従ってください。

<Steps>
  <Step num="1" title="ハンドルを確保する">
    各プレイヤーは宇宙船の指定されたハンドルをしっかり掴む必要があります。左右のスラスターハンドルには、最も連携の取れたプレイヤーを配置してください。
  </Step>
  <Step num="2" title="推進を同期させる">
    直進するためには、左右両方のパイロットがアナログスティックを同時に回転させる必要があります。激しく回すよりも、小さくリズムよく円を描く方が効果的です。
  </Step>
  <Step num="3" title="コントロールされたターンを実行する">
    左に曲がるには、右側のパイロットが推進し、左側のパイロットは固定します。正確な右折を行うには、このプロセスを逆にします。
  </Step>
  <Step num="4" title="安全にドッキングして脱出する">
    着陸プラットフォームに到達したら、安定担当はすぐに周囲の環境を掴んで固定してください。船が完全に固定されるまで、宇宙船のハンドルを離してはいけません。
  </Step>
</Steps>

<Callout type="warning" title="スラスターの不一致">
両方のスラスターパイロットが意思疎通なしに反対方向に引くと、宇宙船は急速に回転します。安定した軌道を維持するために、「左を引いて」や「両方前進」など、推進のタイミングを指示する明確なボイスリーダーを決めましょう。
</Callout>

</Section>

<Section icon={Swords} title="スペースバウンド・スクワッド：攻略と戦略">

スペースバウンド・スクワッドのキャンペーンレベルは、狭い通路、動くレーザー障壁、そして船をコース外へ引き込む重力井戸が組み合わさっています。これらのレベルをうまく攻略するには、防御的なアンカー（固定）と攻撃的な慣性移動を組み合わせる必要があります。

宇宙船を飛ばしている間は、環境そのものが脅威となります。壁に衝突してもすぐに船が壊れることはありませんが、衝撃でプレイヤーがハンドルから振り落とされ、近くの危険物に漂流してしまう可能性があります。

| 障害物 | 危険度 | 対抗策 |
| :--- | :--- | :--- |
| 動くデブリ | 中 | 軌道サイクルがクリアになるまで待ってから発進する |
| レーザー障壁 | 高 | 安定担当をアンカーに配置し、他のメンバーが通過する間保持する |
| 狭い難所 | 高 | スラスター速度を下げ、片腕でのステアリングを使用する |
| 無重力風 | 中 | 横切る前に反対方向に勢いをつける |

これらの課題を克服するために、『Heave Ho 2』にはアンロック可能なサポートアイテムがいくつか用意されています。これらのアイテムは、難しいセクションでチームを支援するために、レベル開始前に選択できます。

  </Card>

  </Card>
