<Section icon={Target} title="ローカル ヒーブホー 2 ゲームシェア セットアップガイド">

ローカルの**ヒーブホー 2 ゲームシェア**セッションの設定は簡単ですが、ホストとゲストの両方の本体で特定の順序で操作を行う必要があります。この方法は、インターネット接続が制限されている、あるいは利用できない場合に最適で、ハードウェア間に直接リンクを作成します。

<Steps>
  <Step num="1" title="ホストセッションの初期化">
    製品版を所有しているメインの本体で、メインメニューから**ゲームシェア**オプションを選択します。そこから**ローカルゲームシェア**を選び、ロビー信号のブロードキャストを開始します。
  </Step>
  <Step num="2" title="ゲスト本体の接続">
    ゲスト側の本体で、ゲーム（またはゲームシェア用クライアント）を開きます。**ゲームシェア**を選択し、次に**ゲームシェアに参加**、最後に**ローカルユーザー**を選択します。本体がアクティブなローカルロビーの検索を開始します。
  </Step>
  <Step num="3" title="アカウントの選択">
    ゲストは使用したいローカルアカウントを選択します。選択すると、画面にホストのロビーが表示されます。決定ボタン（通常は「A」）を押してセッションに参加します。
  </Step>
  <Step num="4" title="確認と開始">
    ゲストがホストのロビーに表示されたら、ホストはスタートを押すことができます。ローカルモードでは、直接的なハードウェア接続を維持するためにオンライン機能が無効になることに注意してください。
  </Step>
</Steps>

ローカルでプレイする場合、遅延は事実上存在しません。これは、正確なタイミングと物理演算に大きく依存するこのゲームにとって極めて重要です。空中で友達の手を掴もうとする時、ミリ秒単位の差が結果を左右します。

| ローカルコンポーネント | ホストの要件 | ゲストの要件 |
| :--- | :--- | :--- |
| **ソフトウェア** | 製品版がインストール済み | ゲームシェア用クライアントまたはデモ版 |
| **接続性** | ローカル通信が有効 | ローカル通信が有効 |
| **プロフィール** | メインの所有者アカウント | 任意のローカルユーザープロフィール |
| **距離** | 10〜15フィート（約3〜4.5m）以内 | 10〜15フィート（約3〜4.5m）以内 |

<Callout type="warning" title="接続のヒント">
ゲスト本体がホストを見つけられない場合は、両方のデバイスが直接のローカル無線信号を妨害する可能性のある5GHz Wi-Fiネットワークに接続されていないか確認してください。機内モードをオン・オフすることで検索がリセットされる場合があります。
</Callout>

  </Card>
  <Card title="ゲストの体験">

  </Card>
  <Card title="プライベートロビー">
    ゲームシェアを基本的なモードに制限する他のゲームとは異なり、このタイトルでは**プライベートオンラインロビー**をホストできます。つまり、ゲームを所有しているフレンドと、ゲームシェア経由で参加しているフレンドを混ぜて遊ぶことができます。

| **セーブデータ** | 永続的 | セッション単位 |

オンラインでゲームシェアを利用してプレイする場合、全員が参加したらゲームチャットウィンドウを最小化しましょう。これによりゲームプレイがフルスクリーンで表示され、難しいチャレンジで小さな足場や障害物を見つけるために不可欠な視界を確保できます。

<Section icon={Swords} title="ゲームモードと物理演算のマスター">

**ヒーブホー 2 ゲームシェア**の接続が確立されると、いくつかの異なるゲームモードにアクセスできます。各モードは、ゲーム独自の2ボタン操作スキームを使用して、協力（または競争）する能力をテストします。左右の手を独立して操作し、トリガーで掴み、アナログスティックで体をスイングさせます。

| モード名 | ゲームプレイの焦点 | プレイヤー数 |
| :--- | :--- | :--- |
| **チュートリアル** | 基本的な動きの学習 | 1-4人 |
| **協力プレイ** | 一緒にゴールを目指す | 2-4人 |
| **チャレンジ** | タイムベースのスキルテスト | 1-4人 |
| **対決** | 対戦型ミニゲーム | 2-4人 |

物理エンジンはこの体験の心臓部です。勢い（モーメンタム）は最大の味方であり、最大の敵でもあります。協力プレイで成功するには、プレイヤーが「人間チェーン」を作る必要があることがよくあります。これには、先頭のプレイヤーがしっかりとしたアンカーポイントを掴み、他のプレイヤーがお互いの手や足を掴む必要があります。

**不可欠な移動テクニック:**
- **スイング**: アナログスティックをリズムよく円を描くように動かして勢いをつけます。
- **トス**: スイングの頂点で手を離し、チームメイトを隙間の向こう側へ投げ飛ばします。
- **アンカー**: 両手で表面をしっかりと掴み、チームの他のメンバーのために安定した土台を作ります。
- **キャッチ**: 片手を空けておき、飛んでくるチームメイトを掴む準備をしておきます。

<Callout type="info" title="物理演算の豆知識">
ヒーブホー 2では、キャラクターの重量は勢いに応じて変化します。スイングしているプレイヤーは、静止しているプレイヤーよりも掴まり続けるのがはるかに困難です。アンカー役のプレイヤーが崖から引きずり落とされないよう、コミュニケーションが鍵となります！
</Callout>

<Section icon={Shield} title="トラブルシューティングとパフォーマンスのヒント">

**ヒーブホー 2 ゲームシェア**システムは非常に安定していますが、特にオンラインセッション中に接続の問題や入力ラグが発生することがあります。1フレームのラグが全員の落下につながる可能性があるこのゲームにおいて、スムーズな体験を維持することは極めて重要です。

| 問題 | 考えられる原因 | 推奨される解決策 |
| :--- | :--- | :--- |
| **ロビーが見つからない** | バージョンの不一致 | 両方の本体を最新パッチに更新する |
| **入力ラグ** | 高いPING値（オンライン） | 有線接続または5GHz Wi-Fiを使用する |
| **切断される** | 信号の干渉 | 本体を近づける（ローカル）またはルーターをリセット |
| **招待に失敗する** | プライバシー設定 | OSの設定で「ゲームへの招待を許可」を有効にする |

ゲームシェア中に「ゴースト」現象やカクつきが発生する場合、多くはホストのアップロード速度が原因です。ホストは実質的にゲームデータをゲストにストリーミングしているため、安定した上り回線が必要です。2026年のオンラインゲームシェアを快適に楽しむには、少なくとも5〜10 Mbpsのアップロード速度を推奨します。
