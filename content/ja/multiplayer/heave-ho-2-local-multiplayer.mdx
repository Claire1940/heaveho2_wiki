<Section icon={Users} title="ローカルマルチプレイ設定の最適化">

ローカルセッションの設定は簡単ですが、協力はスタートボタンを押す前から始まっています。本作は画面分割ではなく「共有スクリーン」視点を採用しているため、全員が同じカメラの視野内にとどまる必要があります。これには「誰も置き去りにしない」という精神が必要です。Nintendo Switch 2では、ゲームシェア機能により、ソフトの所有者1人が近くにある最大3台の本体に対してセッションをホストできるため、ローカルのソファプレイとローカル通信プレイの垣根を取り払うことができます。

<Steps>
  <Step num="1" title="コントローラーの接続">
    ゲームを起動する前に、すべてのゲームパッドが同期されていることを確認してください。PCでは、ネイティブなプラグアンドプレイをサポートするXInputコントローラー（Xbox用など）が推奨されます。
  </Step>
  <Step num="2" title="ゲームモードの選択">
    メインメニューから、キャンペーン（72ステージ）またはショーダウン（56対戦ステージ）を選択します。
  </Step>
  <Step num="3" title="キャラクターのカスタマイズ">
    各プレイヤーはユニークな色と衣装を選択します。この時間を利用して、「アンカー（固定役）」や「スインガー（揺れ役）」などの「役割」を分担しておきましょう。
  </Step>
  <Step num="4" title="グリップの調整">
    チュートリアルエリアに入り、どのトリガーがどちらの手を操作するのかを全員が正しく理解しているか確認します。
  </Step>
</Steps>

<Callout type="warning" title="コントローラーの競合">
旧型のSwitchハードウェアで入力遅延が気になる場合は、Joy-ConとProコントローラーの混用を避けてください。最高の体験を得るためには、4人全員で同じタイプのコントローラーを使用することをお勧めします。
</Callout>

</Section>

<Section icon={Target} title="協力プレイの物理法則をマスターする">

**ヒーブホー 2 ローカルマルチプレイ**の真髄は、キャラクターの左右の手を独立して操作することにあります。従来のプラットフォームゲームとは異なり、ジャンプボタンは存在しません。地形を掴み、自分の体重を利用して遠心力を生み出すことで移動します。これには、ローカルプレイヤー間での緊密な声の掛け合いが不可欠です。

<CardGrid cols={2}>
  <Card title="アンカー戦略">
    1人のプレイヤーが安全な足場をしっかりと掴んで固定され、他のプレイヤーが鎖（チェーン）を形成します。このプレイヤーはセーフティネットの役割を果たすため、チェーンの先頭が安全を確保するまで絶対に手を離してはいけません。
  </Card>
  <Card title="人間ムチ">
    リズムよくスイングすることで、アンカーはチェーンの末端を大きな隙間の向こう側へと「ムチのように」振り出すことができます。最大の飛距離を得るには、弧の頂点で手を離すタイミングが極めて重要です。
  </Card>
  <Card title="交互登り">
    垂直な壁を登る際は、アナログスティックを円を描くように回しながら、左右の手（左・右・左）を交互に掴み直すことで、効率よく壁を登ることができます。
  </Card>
  <Card title="空中キャッチ">
    仲間が落下してしまった場合、フリーの手を持つプレイヤーが腕を伸ばして空中でキャッチすることができます。これには、落下しているプレイヤーも手を広げて掴む準備をしておく必要があります。
  </Card>
</CardGrid>

| アクション | コントローラー入力 | 戦術的な用途 |
| :--- | :--- | :--- |
| **左手で掴む** | L / ZL トリガー | 壁や仲間に体を固定する |
| **右手で掴む** | R / ZR トリガー | 次の掴みポイントへ手を伸ばす |
| **回転** | 左アナログスティック | スイングの勢い（慣性）をつける |
| **離す** | トリガーを離す | 自分自身や仲間を放り投げる |
| **アシストドローン** | 自動（失敗が続いた場合） | 極めて困難な難所をスキップする |

<Callout type="success" title="高度なテクニック">
アンロックアイテム「イマ・ロック（Ima Rock）」を使用すると、キャラクターを空中の一点に固定できます。これにより、仲間が掴まって勢いをリセットするための、一時的な「動かない足場」を作り出すことができます。
</Callout>

</Section>

<Section icon={Trophy} title="アンロック要素とコラボコンテンツ">

**ヒーブホー 2 ローカルマルチプレイ**の最もやりがいのある要素の一つが、豊富なカスタマイズシステムです。オプションの目標を達成したり、隠されたギフトを見つけたりすることで、Devolver Digitalの他の大ヒット作との豪華なコラボスキンを含む、さまざまなスキンをアンロックできます。これらのスキンは見た目を変えるだけですが、混沌としたセッションにさらなる個性を加えてくれます。

| アイテムタイプ | アンロック方法 | 例 |
| :--- | :--- | :--- |
| **コラボスキン** | 隠されたシルエットを見つける | Among Us, Cult of the Lamb, Balatro |
| **サポートギア** | リワードトラックを完了する | アームスプレー, 傘, ポップガン |
| **ギフト** | ステージ内のプレゼントを回収する | 帽子, メガネ, ユニークなカラー |
| **実績** | 特定のタスクを達成する | 「ギフト中毒」, 「タイムチャンピオン」 |

<Callout type="tip" title="アンロックのヒント">
「アームスプレー（Arm Spray）」は、キャラクターの手が届く範囲を広げてくれるため、ローカルプレイで特に役立ちます。落下する仲間を救ったり、通常は4人のチェーンが必要な遠くの足場に手を伸ばしたりするのが非常に簡単になります。
</Callout>
