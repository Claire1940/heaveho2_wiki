<Section icon={Target} title="初心者向けゲームプレイと進行ガイド">

ヒーブホー 2の進行は、8つのテーマに沿ったアドベンチャーをクリアすることで構成されています。各ワールドには、プレイヤーに動き方や相互作用の再考を促すユニークなメカニズムが導入されています。従来のプラットフォームゲームとは異なり、ジャンプボタンはありません。すべての動きは、独立して制御される2本の腕で表面を掴み、勢い（モメンタム）を利用して自分自身やチームメイトを隙間の向こう側へ飛ばすことによって行われます。

<Steps>
  <Step num="1" title="独立した手の操作をマスターする">
    コントローラーの各トリガーは特定の手に対応しています。左手で表面を掴みながら、体を回転させて右手で次のポイントに手を伸ばす練習をしましょう。
  </Step>
  <Step num="2" title="確実なアンカーを確立する">
    チームが長いスイングを試みる前に、少なくとも1人のプレイヤーが安定した足場にしっかりと固定されていることを確認してください。この「アンカー」プレイヤーは、人間チェーン全体のセーフティネットとして機能します。
  </Step>
  <Step num="3" title="人間チェーンを構築する">
    手や体を掴んでプレイヤー同士を繋ぎます。チェーンが長いほど遠くの足場に届きますが、安定させて回転させるのは非常に難しくなります。
  </Step>
  <Step num="4" title="勢いをコーディネートする">
    同期した円運動を行って速度を上げます。「呼びかけ役」がカウントダウン（3、2、1、離して！）を行い、スイングのピークで先頭のプレイヤーが発射されるようにします。
  </Step>
  <Step num="5" title="回復ツールを活用する">
    プレイヤーが落下した場合、残りのチームはアンカーを維持する必要があります。**イマロック（Ima Rock）**のようなアンロック可能なアイテムを使用して、救助されたチームメイトのために一時的な空中足場を作成しましょう。
  </Step>
</Steps>

<Callout type="warning" title="よくある間違い">
すべてのプレイヤーが同時に手を離さないようにしてください。固定されたアンカーポイントがないと、チーム全員が落下し、現在のセクションを最初からやり直すことになります。
</Callout>

</Section>

<Section icon={Swords} title="操作方法と高度なスイングテクニック">

操作のシンプルさの裏には、物理エンジンの深みがあります。ゲームをマスターするには、基本的なクライミングから「モメンタム移譲」や「空中キャッチ」などの高度な操作へと移行する必要があります。**ヒーブホー 2 ローンチトレーラー**では、プレイヤーが精密な精度で移動する足場をナビゲートする高速セグメントがいくつか紹介されました。

| アクション | コントローラー入力 | 高度なテクニック |
| :--- | :--- | :--- |
| **移動/回転** | 左アナログスティック | 精密な操作には小さな円、加速には大きな円を描く |
| **左手のグリップ** | 左トリガー (L2/LT) | 右手で次を探す間、アンカーとして保持する |
| **右手のグリップ** | 右トリガー (R2/RT) | 間違ったチームメイトを掴まないようインジケーターを確認する |
| **味方を投げる** | スイングを溜めて離す | 投げられるプレイヤーの空いている手をターゲットに向ける |
| **表面を登る** | グリップを交互に入れ替える | 片手ずつ移動する。両手を同時に離してはいけない |
| **味方をキャッチ** | 接触時にグリップを保持 | 早めに手を開き、衝撃を吸収するために安定を保つ |

<CardGrid cols={2}>
  <Card title="アンカー戦略">
    最も強いプレイヤーを永久アンカーに指名しましょう。彼らが環境に固定されている間、他のプレイヤーが探索したりオブジェクトを運んだりすることで、チームの進行状況が失われないようにします。
  </Card>
  <Card title="リズムのカウント">
    4人協力プレイでは、ボイスチャットを使って回転を同期させます。共通のリズムを持つことで、プレイヤーが反対方向に引っ張り合って勢いを打ち消してしまう「ゴム」効果を防ぐことができます。
  </Card>
</CardGrid>

<Callout type="tip" title="プロのテクニック">
小さく狭い足場に着地するときは、頭をターゲットの少し上に向けます。これにより、手が自然に表面に落ちて掴みやすくなります。
</Callout>

<Section icon={Sparkles} title="マルチプレイヤーモードと協力プレイの戦略">

ヒーブホー 2は、友達と遊ぶための様々な方法を提供しています。キャンペーンがメインの魅力ですが、対戦型の **ショーダウン（Showdown）** モードでは、56のレベルでテンポの速いライバル争いを楽しめます。Nintendo Switch 2バージョンは特に **GameShare** 機能が注目されており、ゲーム所有者1人が、ゲームを持っていない他の3人のプレイヤーのためにセッションをホストすることができます。

| モード | プレイヤー数 | プラットフォーム | 主要な戦略 |
| :--- | :--- | :--- | :--- |
| **オンライン協力** | 2–4人 | すべて | 単語での呼びかけ（持て、振れ、離せ）を使用する |
| **ローカル協力** | 2–4人 | すべて | キャラクターと同じ順序でプレイヤーを座らせる |
| **ショーダウン** | 2–4人 | すべて | 回転ラウンドではスピードよりも生存に集中する |
| **GameShare** | ホスト1 + 3人 | Switch 2 | ホストはGameChat経由でセッションを開始する |
| **Remote Play** | 2–4人 | PC | 安定した低遅延の有線接続が必要 |

<Callout type="info" title="ネットワーク要件">
Nintendoバージョンでショーダウンや協力プレイを含むオンラインマルチプレイヤー機能を利用するには、有効な **Nintendo Switch Online** サブスクリプションが必要です。
</Callout>

</Section>
