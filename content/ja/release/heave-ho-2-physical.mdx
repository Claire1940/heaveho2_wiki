<Section icon={Swords} title="操作方法とスイングメカニクスのマスター">

『ヒーブホー 2』の核心は、その「両手」操作システムにあります。ボタンを押してジャンプする従来のプラットフォームゲームとは異なり、360度の環境をナビゲートするには、手動で掴み、回転し、離す必要があります。

<Steps>
  <Step num="1" title="個別の手の操作">
    左右のトリガー（または指定されたキー）を使用して、それぞれの手を独立して閉じます。片手で表面を掴みながら、もう片方の手で次へ手を伸ばす練習をしましょう。
  </Step>
  <Step num="2" title="勢いをつける">
    左アナログスティックを使用してキャラクターを回転させます。大きな隙間を越えるには、円を描くようにスイングし、回転ごとに速度を上げてから手を離す必要があります。
  </Step>
  <Step num="3" title="人間の鎖を作る">
    協力プレイでは、チームメイトの手や体を掴みます。後ろのプレイヤーが**アンカー**となり、前のプレイヤーが次の場所を掴みます。
  </Step>
  <Step num="4" title="完璧なリリース">
    スイングの外側への弧が頂点に達した瞬間に手を離します。離すのが早すぎたり遅すぎたりすると、垂直に落下したり、後ろに飛んでしまったりします。
  </Step>
</Steps>

| アクション | コントローラー入力 | プロのテクニック |
| :--- | :--- | :--- |
| **左手で掴む** | Lトリガー / L1 | 回転の軸として使用 |
| **右手で掴む** | Rトリガー / R1 | 遠くのチームメイトに手を伸ばす |
| **回転** | 左アナログスティック | チームのリズムに合わせて動く |
| **投げる** | 勢いをつけて離す | ターゲットの少し上を狙う |
| **キャッチ** | 接触時に掴むホールド | ぶつかる前に早めに手を開く |

<Callout type="success" title="高度な戦術">
アンロックアイテムの**イマ・ロック（Ima Rock）**を使用して、空中で自分を固定しましょう。これにより、ジャンプをミスしたチームメイトのための一時的な静止足場として機能できます。
</Callout>

</Section>

<Section icon={Target} title="ワールドガイドとキャンペーン攻略">

キャンペーンは**8つのテーマ別ワールド**と、約**72の協力レベル**で構成されています。各ワールドには、チームが移動戦略を適応させる必要があるユニークなギミックが登場します。ダウンロード版でも**ヒーブホー 2 パッケージ版**でも、レベルの進行状況は同じです。

| ワールド名 | 順番 | 主要ギミック | 主な目的 |
| :--- | :--- | :--- | :--- |
| **フライング・ハイ** | 1 | スキーリフト / ゴンドラ | 動く輸送プラットフォームを移動する |
| **マナー・ミステリー** | 2 | 隠された鍵 | 出口のドアを開けるために鍵を回収する |
| **スペースバウンド・スクワッド** | 3 | 無重力 | 宇宙船を操縦し、漂流を制御する |
| **シェフズ・オーダー** | 4 | 大きな食材 | 動く皿の上に食べ物を積み上げる |
| **シップレックド** | 5 | インタラクティブな障害物 | スイング中に環境トラップを避ける |
| **トランスフォラボ** | 6 | 精密機器 | チームで実験室の機械を操作する |
| **カスケード・ショー** | 7 | 動く要素 | 変化する地形でジャンプのタイミングを計る |
| **メディーバル・トラベル** | 8 | 王座の輸送 | 王様を安全にゴールまで運ぶ |

<Callout type="info" title="レベルで行き詰まったら？">
同じセクションで何度も失敗すると、**キャットドローン（Cat Drone）**アシストが登場することがあります。このツールは、苦戦しているプレイヤーをゴールまで引っ張ってくれるため、先に進んでから後でオプションのチャレンジに戻ることができます。
</Callout>

各レベルには、**タイマー**、**ミステリアス・ハンド**、**隠されたレシピ**などのオプション目的もあります。これらを完了することは、クロスオーバースキンやサポートアイテムの全リストをアンロックするために不可欠です。

</Section>

<Section icon={Sparkles} title="マルチプレイモードとGameShare機能">

『ヒーブホー 2』はソーシャルプレイ向けに設計されており、協力モードと対戦モードの両方を提供しています。Nintendo Switch 2版の追加により、ゲームを持っていない友人とプレイする画期的な方法が導入されました。

<CardGrid cols={2}>
  <Card title="協力キャンペーン">
    - **2〜4名**のローカルまたはオンラインプレイ。
    - 協力してゴールを目指す。
    - 共有スクリーンカメラシステム。
  </Card>
  <Card title="ショーダウンモード">
    - 対戦形式のチャレンジ。
    - レース用の**56の専用マップ**。
    - サバイバルや「ハンドルを掴め」などの目標。
  </Card>
</CardGrid>

**Nintendo Switch 2 の GameShare** 機能はこの続編の目玉です。ゲームの所有者1人がセッションをホストすることで、ローカルまたはオンラインのGameChatシステムを介して、他の最大3人のプレイヤーが無料で参加できます。これにより、大人数の友人グループでも遊びやすくなっています。

| モード | プレイヤー数 | オンライン対応 | 必要条件 |
| :--- | :--- | :--- | :--- |
| **ローカル協力** | 2–4 | いいえ | 追加のコントローラー |
| **オンライン協力** | 2–4 | はい | NSO / Steam アカウント |
| **ローカル対戦** | 2–4 | いいえ | 共有スクリーン |
| **GameShare** | 1 ホスト + 3 | はい | **Switch 2 専用** |

<Callout type="warning" title="コミュニケーションが鍵">
オンラインモードでは、内蔵のボイスチャットや外部アプリを使用してください。物理演算の調整には、「せーの、3、2、1、離して！」といった正確なタイミングが必要で、口頭での合図なしではほぼ不可能です。
</Callout>

</Section>
