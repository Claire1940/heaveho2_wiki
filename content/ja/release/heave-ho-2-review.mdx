<Section icon={Target} title="物理演算のマスター：ゲームプレイメカニクス">

**ヒーブホー 2**の核心は、一見シンプルに見えて奥が深い操作方法にあります。プレイヤーが操作するのは、2本の長い伸縮自在の腕を持つ「頭」だけです。ジャンプボタンはなく、掴む、揺れる、投げるアクションを駆使してゴールを目指します。今作では、チーム全員の息を合わせる必要がある「ポップガン」や「ドローン」など、より複雑なインタラクションが追加されています。

**動画の見どころ：**
- 基本的な腕の回転とグリップのタイミング。
- 遠投のために最大の遠心力を生み出す方法。
- 空中での新しいサポートアイテム「イマロック」の実演。

キャンペーンは**8つのユニークなワールド**に拡張され、それぞれに異なる環境トラップや移動ギミックが導入されています。難易度曲線は非常にうまく調整されており、雪山の氷の斜面から始まり、最終的には中世の王座を運ぶという高難度のミッションへと繋がっていきます。

| ワールド名 | 基本ギミック | 主な難易度要因 | クリア目標 |
| :--- | :--- | :--- | :--- |
| **Flying High（フライングハイ）** | スキーリフト / ゴンドラ | 動く足場 | 山頂に到達する |
| **Manor Mysteries（マナーミステリー）** | 隠された鍵 | オブジェクトの受け渡し | 出口をアンロックする |
| **Spacebound Squad（無重力スクワッド）** | 無重力 | 慣性コントロール | 宇宙船を操縦する |
| **Chef's Orders（シェフのオーダー）** | 食材の積み重ね | 巨大なオブジェクトの物理挙動 | 料理を盛り付ける |
| **Shipwrecked（難破船）** | 水中の物理挙動 | 浮遊する足場 | 遺跡を通り抜ける |
| **Transfolab（トランスフォラボ）** | 連動デバイス | 正確なタイミング | 研究所のパズルを解く |

**Medieval Travels**ワールドは、チームの連携力を試す究極のテストです。プレイヤーは王様を乗せた王座を、ひっくり返さないように運ばなければなりません。誰か一人が引っ張りすぎると王座が反転し、そのセクションは即やり直しになるため、完全に同期した動きが求められます。

<Section icon={Swords} title="対戦プレイ：ショーダウンモード">

協力プレイが本作の最大の魅力ですが、**ショーダウン（Showdown）モード**では、ハチャメチャな対戦を楽しむことができます。**56のランダムな対戦レベル**が用意されており、レース、サバイバル、そして「ハンドル奪い合い」などのチャレンジで競い合います。最初に5勝したプレイヤーまたはチームが勝者となります。

| モードの種類 | 目標 | 最適な戦略 |
| :--- | :--- | :--- |
| **レース** | 誰よりも早くゴールに到達する | リスク覚悟の遠投スロー |
| **サバイバル** | 最後まで生き残る | 姿勢を低く保ち、2つのアンカーを維持する |
| **ハンドルグラブ** | ターゲットのハンドルを保持する | 其他のプレイヤーの手を振り払って妨害する |
| **チームショーダウン** | 2対2のチーム戦目標 | 1人がアンカーとなり、もう1人が走る |

ショーダウンのレベルは、1回あたり60秒未満で終わるスピーディーな設計になっています。ここでは物理エンジンが真価を発揮し、対戦相手を掴んでプラットフォームから引きずり落とすといったアクティブな妨害が可能です。ローカルの協力プレイでは、爆笑必至（そして時に友情破壊を招く）「裏切り」が頻発するでしょう。

<Callout type="info" title="オンラインプレイ">
ショーダウンモードはオンラインマッチングに完全対応しています。PCでプレイする場合は、わずかな遅延でもレース中の重要な掴み損ねに繋がるため、安定したインターネット環境を確保することをおすすめします。
</Callout>

</Section>
