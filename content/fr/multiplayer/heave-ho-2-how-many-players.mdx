<Section icon={Swords} title="Modes Multijoueurs : Co-op vs Showdown">

Heave Ho 2 s'appuie sur les bases du premier opus en proposant deux façons distinctes de jouer avec vos amis. Comprendre la différence entre les modes Campagne et Showdown est essentiel pour choisir l'expérience adaptée à la taille de votre groupe.

### Campagne Coopérative
Dans la campagne principale, 2 à 4 joueurs travaillent ensemble pour naviguer à travers 72 niveaux répartis dans huit mondes thématiques. Les mécaniques nécessitent de la synchronisation ; si une personne lâche prise trop tôt, toute la chaîne peut s'effondrer. Ce mode est idéal pour les groupes qui aiment la résolution d'énigmes et le "chaos planifié".

### Showdown Compétitif
Si votre groupe préfère la rivalité, le mode Showdown propose 56 niveaux conçus pour la compétition. Les joueurs font la course les uns contre les autres ou s'affrontent dans des défis de survie. Le premier joueur ou la première équipe à atteindre cinq victoires est couronné vainqueur.

<Section icon={Target} title="Guide de Configuration : Comment se Connecter avec des Amis">

La configuration d'une session multijoueur dans Heave Ho 2 varie légèrement selon la plateforme choisie. Suivez ces étapes pour garantir une connexion stable pour votre groupe de 2 à 4 joueurs.

<Steps>
  <Step num="1" title="Sélectionnez Votre Mode">
    Depuis le menu principal, choisissez entre **Jeu Local** ou **Jeu en Ligne**. Si vous jouez localement, assurez-vous que toutes les manettes sont synchronisées avec le système avant de commencer.
  </Step>
  <Step num="2" title="Invitez vos Coéquipiers">
    En Jeu en Ligne, l'hôte peut envoyer des invitations via la liste d'amis Steam ou le menu d'amis Nintendo Switch. Sur la Nintendo Switch 2, vous pouvez utiliser **GameChat** pour lancer une session.
  </Step>
  <Step num="3" title="Personnalisation du Personnage">
    Une fois que tous les joueurs ont rejoint le lobby, chacun peut personnaliser son personnage. C'est le moment idéal pour équiper les **skins crossover** que vous avez débloqués, comme l'équipier d'Among Us.
  </Step>
  <Step num="4" title="Choisissez un Monde">
    L'hôte sélectionne le monde de départ. Pour les nouveaux groupes, **Flying High** est recommandé pour apprendre les mécaniques de base de balancement et de saisie.
  </Step>
</Steps>

### GameShare Nintendo Switch 2
La version Nintendo Switch 2 introduit une fonctionnalité GameShare révolutionnaire. Cela permet à un seul propriétaire de Heave Ho 2 de partager le jeu avec jusqu'à trois autres joueurs à proximité ou en ligne, ce qui signifie qu'une seule personne de votre groupe doit acheter le jeu pour profiter de l'expérience complète à 4 joueurs.

<Callout type="warning" title="Important">
Lorsque vous utilisez le Steam Remote Play Together, assurez-vous que l'hôte dispose d'une connexion Internet filaire haut débit. Comme le jeu repose sur un timing physique précis, même une latence mineure peut faire rater les prises aux joueurs.
</Callout>

</Section>
